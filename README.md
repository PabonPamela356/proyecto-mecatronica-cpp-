# Sistema de alumnos

Este es un programa hecho en C++ que se conecta a una base de datos MySQL para gestionar alumnos.

## Requisitos

* MySQL instalado
* Compilador de C++ (Code::Blocks o similar)

## La instalación

1. Abrir MySQL Workbench
2. Ejecutar el archivo `database.sql`
3. Compilar el archivo `main.cpp`
4. Ejecutar el programa

## Funciones del sistema

* Registrar alumno
* Mostrar lista de alumnos
* Eliminar alumno
* Actualizar nota
* Mostrar Top 5 mejores notas
* Mostrar alumnos en riesgo (nota menor a 4)
* Buscar alumno por ID

## Notas

* La conexión se realiza a `localhost`
* Usuario: root (la contraseña es yato_777@)
* Base de datos: nexus_academy
